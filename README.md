# Team 4 Backend Application
Team 4 Backend Application Feb/March 2026

## Getting Started

### Clone and Setup
```bash
git clone https://github.com/GarethNelsonKainos/team4-back-app.git
cd team4-back-app
npm install
```

## Testing
```bash
# Run tests in watch mode
npm test

# Run tests once
npm run test:run

# Run tests with coverage
npm run test:coverage

# Open test UI
npm run test:ui
```
